# gatsby-typescript-boilerplate
Opinionated [Gatsby](https://www.gatsbyjs.org/) V2 with [TypeScript](https://www.typescriptlang.org/) boilerplate. 

![Gatsby + TypeScript](src/images/gatsby-typescript.jpg)

## Includes
* Gatsby V2
* TypeScript
* Prettier
* Stylelint
* TSLint

## Getting started
* Clone repo
* npm install
* npm run develop
